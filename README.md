## Installation
- clone the repository
```bash
$ git clone https://github.com/Pazl27/Asteroids.git
```
- make sure you have go installed
```bash
$ go --verions
```
- run the game with go
````bash
$ go run cmd/main.go
```
- this step can take a while, as go will download all the dependencies


## Controls 
- `W` to move forward
- `A` to rotate left
- `D` to rotate right
- `S` to stop
- `Space` to shoot
